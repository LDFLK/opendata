# Getting Started

## Prerequisites

- Ruby (version 2.5.0 or higher recommended)
- Bundler
- Jekyll
- Git


1. **Fork & Clone the Repository**

```bash
git clone https://github.com/your-username/opendata.git
cd opendata
```

2. **Install Dependencies**

```bash
bundle install
```

3. **Run the Server**

```bash
bundle exec jekyll serve
```

Open `http://localhost:4000` in your browser to see the site.
